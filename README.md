# Python数据科学指南

本指南旨在为管理/商学院学生和数据科学初学者提供一份全面且友好的学习资料。

## 本指南的目标

本指南旨在帮助读者快速入门数据科学领域，特别是掌握利用 Python 进行数据处理、分析和可视化的基本技能。我们相信，数据科学不仅仅是技术人员的专利，更是现代商业人士必备的技能之一。

本指南主要包含以下几个部分：

### 第一篇：Python数据科学入门

本篇主要面向零基础的初学者，详细介绍数据科学环境的搭建和基础工具的使用。内容包括：

*   **核心概念**: 介绍 Python、Anaconda 和 Jupyter Notebook 的基本概念及其在数据科学中的作用。
*   **开发环境**: 概览常用的 Python IDE，如 PyCharm、VS Code，以及AI辅助编程工具。
*   **安装指南**: 提供在主流操作系统上安装和配置 Anaconda 和 Jupyter Notebook 的详尽步骤。
*   **Anaconda基础**: 指导读者学习 Anaconda 的基本用法，包括环境管理和包的安装。
*   **PIP包管理**: 介绍 `pip` 包管理器的使用，以及如何配置国内镜像源（如清华TUNA）来加速下载。
*   **Jupyter Notebook快速入门**: 通过一个简单的实例，引导读者完成第一个Jupyter Notebook项目。
*   **学习资源**: 提供一系列精选的学习资源链接，以供读者进一步深入学习。
*   **总结**: 对本篇内容进行归纳总结。

### 第二篇： Python 数据科学进阶

第二篇为进阶学习篇，基于实践驱动，带你从数值计算与表格数据处理到可视化展示完成一个端到端的小流程。主要内容（已整理为 Lesson 2 系列）包括：

*   **NumPy**：介绍 `ndarray`、向量化运算、广播、形状变换与常用数值函数，是高效数值计算的基础。
*   **Pandas**：讲解 `Series` 与 `DataFrame` 的创建、索引/切片、分组聚合、缺失值处理与数据清洗/转换，面向表格数据的日常分析工作流。
*   **Matplotlib**：掌握绘图的面向对象 API、常见图表（折线、柱状、直方、散点、箱线图、饼图）以及图表定制与保存方法，用于将分析结果可视化表达。
*   **Toy case（端到端示例）**：把 NumPy、Pandas 与 Matplotlib 串联起来，展示从数据加载、处理到可视化的完整小案例，帮助读者把工具串成可复用的流程。
*   **FAQ 与常见问题排查**：收录在使用 NumPy/Pandas/Matplotlib 时常见的问题（如中文字体、数据类型、性能陷阱）与对应的解决方案。
*   **术语表与参考资源**：整理常用概念、函数与延伸阅读链接，便于查阅与后续深入。

第二篇侧重实践与可复现工作流，并配套示例代码与静态图表资源（见课程内链接），建议读者在 Jupyter Notebook 中边看边运行示例以加深理解。

### 未来内容规划

在完成第一、二篇的基础和进阶内容后，我们计划继续拓展本指南的深度和广度，将陆续更新以下内容：

*   **机器学习与数据挖掘算法**: 系统介绍主流的机器学习和数据挖掘算法，如回归、分类、聚类、关联规则等。
*   **实战案例**: 结合真实或模拟的数据集，提供端到端的编程案例，帮助读者将理论知识应用于实践。
*   **商业应用**: 探讨数据科学在市场营销、金融风控、运营管理等不同商业场景下的实际应用，并结合案例进行分析。
